### Настройки-Список сайтов-Добавление
Название: 
Проверка добавления сайтов

Предусловия:
1. Открыть окно "Настройки"
2. Открыть вкладку "Список сайтов"
3. Нажать на кнопку '+'
4. Заполнить поле (см. таблицу ниже, столбец "Вводимое значение")
5. Нажать на кпонпку "Сохранить"
6. Запустить приложение
7. Открыть браузер  и ввести адрес сайта из списка

Ожидаемый результат:

| Вводимое значение          | Ожидаемый результат |
| ------------- |:-------------:|
| Корректный URL-адрес сайта (Например, www.youtube.com)      | URL-адрес сайта сохранится  в списке, страница сайта блокируется приложением
| Некорректный URL-адрес сайта (Например, youtube)  | Отобразится сообщение об ошибке 'Некоррекный адрес' , кнопка "Сохранить" неактивна, запуск приложения невозможен| 
| <Оставить поле пустым>    | Сохранится пустая строка ввода
|  Дубликаты адресов   | Повторяющиеся адреса сайтов сохранятся  в списке, приложение блокирует данные страницы сайтов
|  Вставить горячими клавишами(Ctrl+V) скопированный корректный адрес сайта (Например, https://vk.com/) | Адрес сайта сохранится  в списке, страница сайта блокируется приложением
| Список сайтов (Например, www.youtube.com, www.ok.ru) | URL-адрес сайтов сохранятся в списке, страницы сайтов блокируются приложением

### Настройки-Список сайтов-Удаление
Название: 
Проверка удаления сайта из списка

Предусловия:
1. Открыть окно "Настройки"
2. Открыть вкладку "Список сайтов"
3. Добавить один или несколько сайтов (см. Тест-кейс- Настройки-Список сайтов-Добавление)
4. Убедиться, что чекбокс "Белый список" снят

| Шаги           | Ожидаемый результат |
| ------------- |:-------------:|
| 1. Нажать на кнопку 'х'  |   Выбранные сайты удалятся из списка|
| 2. Открыть браузер и ввести URL-адреса удаленных из списка сайтов  |  Страницы сайтов открываются браузером, функция блокировки недоступна |

### Настройки-Расписание-Добавление
Название: 
Проверка добавления расписаний

Предусловия:
1. Открыть окно "Настройки"
2. Открыть вкладку "Расписание"
3. Нажать на кнопку '+'
4. Заполнить поле (см. таблицу ниже,столбец "Вводимое значение")
5. Нажать на кпонпку "Сохранить"
6. Запустить приложение
7. Перейти во вкладку "Расписание"

Ожиаемый результат:

| Вводимое значение          | Ожидаемый результат |
| ------------- |:-------------:|
| Корректное время и день недели (Например, <12:00><00:00>, Вт, Чт )  | Время и дни недели установлены, отобразится окно с информацией 'До блокировки осталось чч-мм' или 'Нет расписания на сегодня/Следующая блокировка через ...дней'; Автоматически запустится блокировка в соотвествие с выбранным временем; По истечению заданного времени разблокируется |
| Некорректное время (Например,<70:35><45:20>) | Строка ввода останется пустой
| Корректное время и неустановленные дни недели  |Время установлено, дни недели неактивны; отобразится окно с информацией "Нет расписания"
| Список расписаний (Например, <12:00><00:00>, Пн, Пт;<16:00><18:00>, Вт, Чт)| Время и дни недели установлены, отобразится окно с информацией 'До блокировки осталось чч-мм/Следующая блокировка:чч:мм-чч:мм' или 'Нет расписания на сегодня/Следующая блокировка через ...дней'; Автоматически запустится блокировка в соотвествие с выбранным временем; По истеречению заданного времени разблокируется   |
